**Mistake Tracker** is a lightweight Python CLI tool designed to help students and self-learners track their learning errors. By recording and analyzing mistakes, you can identify weak points and improve efficiently.

* **No Dependencies**: Uses only Python standard libraries (Zero `pip install` needed).
